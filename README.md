# Gesture Controlled LED Blinking 💡✋

## 📌 Project Overview

The **Gesture Controlled LED Blinking** project demonstrates how hand gestures can be used to control electronic devices. In this project, a gesture detection system is used to recognize hand movements and send signals to an **Arduino microcontroller**, which then controls the blinking of an LED.

This project highlights the concept of **human-computer interaction using gesture recognition**, making device control more intuitive and interactive.

---

## 🎯 Objectives

* To develop a simple **gesture-based control system**
* To demonstrate **Arduino-based hardware interaction**
* To explore **gesture recognition for automation**
* To create an interactive way to control LEDs without physical switches

---

## ⚙️ Components Used

* Arduino Uno
* LED
* Resistor (220Ω)
* Gesture Sensor / Camera module
* Jumper Wires
* Breadboard

---

## 🔧 Working Principle

The system detects hand gestures using a **gesture sensor or camera module**. The detected gesture is processed and interpreted as a command.

Based on the recognized gesture:

* The Arduino receives the signal.
* It executes the programmed instruction.
* The LED responds by **turning ON, OFF, or blinking**.

This creates a **touchless control system** where gestures replace physical switches.

---

## 🚀 How to Run the Project

1. Connect the **Arduino, LED, resistor, and gesture sensor** according to the circuit diagram.
2. Open the Arduino IDE.
3. Upload the `gesture_led.ino` code to the Arduino board.
4. Power the Arduino.
5. Perform gestures in front of the sensor to control the LED.
